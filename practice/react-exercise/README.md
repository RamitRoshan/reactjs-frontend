# React Exercises :

 
1.	Create a React component that contains a form where the user enters a GitHub username. <br> On form submission, fetch and display the user’s profile information (such as name and avatar). <br>
Primary API: https://api.github.com/users/{username} <br>
Alternate APIs: <br>
	•	GitLab Users API: https://gitlab.com/api/v4/users?username={username}  
	•	Bitbucket Users API: https://api.bitbucket.org/2.0/users/{username}

<br>

2.	Build a React application with a dropdown form that lists countries fetched from a public API. <br> When a country is selected, fetch and display the capital and population of the selected country. <br>
Primary API: https://restcountries.com/v3.1/all  <br>
Alternate APIs: <br>
	•	World Bank Countries API: https://api.worldbank.org/v2/country?format=json <br>
	•	GeoNames Country Info API: http://api.geonames.org/countryInfoJSON (requires username)

<br>

3.	Develop a React component with a form where the user enters a city name. <br> On form submission, fetch and display the current weather information for that city. <br>
Primary API: https://api.openweathermap.org/data/2.5/weather  <br>
Alternate APIs:  <br>
	•	WeatherAPI: https://api.weatherapi.com/v1/current.json   <br>
	•	Open-Meteo: https://api.open-meteo.com/v1/forecast

<br>

4.	Create a React component with a search form that allows users to enter a movie title.  <br> Fetch matching movies from a movie database API and display the list of movie titles.
Primary API: https://www.omdbapi.com/  <br>
Alternate APIs: <br>
	•	The Movie Database (TMDB): https://api.themoviedb.org/3/search/movie  <br>
	•	TVMaze Search API: https://api.tvmaze.com/search/shows?q={query}

<br>

5.	Design a React component with a form input where the user enters a user ID. <br> When the ID changes, fetch and display the corresponding user’s details such as name and email. <br>
Primary API: https://jsonplaceholder.typicode.com/users/{id}  <br>
Alternate APIs: <br>
	•	ReqRes Users API: https://reqres.in/api/users/{id}  <br>
	•	DummyJSON Users API: https://dummyjson.com/users/{id}
